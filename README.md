# redmine-max-notifier

Сервис-интегратор для отправки событийных оповещений из Redmine в мессенджер MAX.

## Возможности

- Приём webhook'ов от Redmine
- Поллинг событий через REST API (для тех, что не отдаются webhook'ом — например, приближение `due_date`)
- Маршрутизация событий: задача → проект → чат/пользователь MAX
- Шаблонизация сообщений (Jinja2)
- Расширяемая архитектура для добавления новых типов событий

## Источники событий

- Создание новой задачи
- Изменение статуса
- Добавление комментария
- Приближение/истечение срока (`due_date`)
- Закрытие задачи

## Стек

Python 3.12+, FastAPI, httpx, Pydantic v2, SQLAlchemy 2.0, APScheduler, Jinja2.
Управление зависимостями — uv. Линтинг — Ruff. Типизация — mypy.
